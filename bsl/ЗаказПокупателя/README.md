# Заказы покупателей (GetCOrders)

HTTP-сервис для выдачи данных документа **ЗаказПокупателя** (Документ.ЗаказПокупателя) в JSON-формате с полями `data` и `errors`.

**Функция в конфигураторе:** `GetCOrders`. Имя шаблона URL должно совпадать с именем функции.

---

## Текущее состояние

- Создан каркас модуля и единый формат ответа.
- Логика выборки и фильтрации будет добавляться поэтапно.

## Планируемые разделы данных

- Реквизиты шапки (номер, дата, контрагент, организация, ответственный, суммы, НДС).
- Табличные части: Товары, Услуги, Материалы, ВозвратнаяТара, СоставНабора.
- Расширенный режим `full` для детальной выдачи.
