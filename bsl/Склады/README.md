# Склады (GetWarehouses)

HTTP-сервис возвращает список складов (Справочник.Склады) в виде единообразного JSON с полями **data** и **errors**.

**Функция в конфигураторе:** `GetWarehouses`. Имя шаблона URL должно совпадать с именем функции.

**Ответ:** JSON с полями `data` (массив элементов с полями Код, Наименование) и `errors` (массив строк ошибок).

---

## Работа с параметрами

Параметры передаются через **ПараметрыЗапроса** или **ПараметрыURL**.

| Параметр | Назначение |
|----------|------------|
| **code** | Подстрока в коде склада (ПОДОБНО). Если не указан — ограничение по коду не применяется. |
| **name** | Подстрока в наименовании склада (ПОДОБНО). Если не указан — ограничение по наименованию не применяется. |

В конфигураторе нужно задать шаблон URL, соответствующий имени функции **GetWarehouses**, либо переименовать функцию под существующий шаблон.
