# Номенклатура (GetMaterials)

HTTP-сервис возвращает элементы справочника **Номенклатура** (Справочник.Номенклатура) в виде дерева по родительским группам. Учитываются только элементы (не группы), не помеченные на удаление. Ответ — единообразный JSON с полями **data** и **errors**.

**Функция в конфигураторе:** `GetMaterials`. Имя шаблона URL должно совпадать с именем функции.

**Ответ:** JSON с полями `data` (массив узлов дерева: группы с полями Код, Наименование, ЭтоГруппа=true, Дети; элементы с Код, Наименование, Артикул, ЭтоГруппа=false, Услуга, ЕдиницаИзмерения, ВидНоменклатуры, НоменклатурнаяГруппа, ОсновнойПоставщик) и `errors` (массив строк ошибок).

---

## Работа с параметрами

Параметры передаются через **ПараметрыЗапроса** или **ПараметрыURL**.

| Параметр | Назначение |
|----------|------------|
| **code** | Подстрока в коде номенклатуры (ПОДОБНО). |
| **name** | Подстрока в наименовании номенклатуры (ПОДОБНО). |
| **group** | Подстрока в наименовании родительской группы (ПОДОБНО). Фильтрует по непосредственному родителю. |
| **type** | Подстрока в виде номенклатуры (ПОДОБНО). |
| **ngroup** | Подстрока в номенклатурной группе (ПОДОБНО). |
| **article** | Подстрока в артикуле (ПОДОБНО). |
| **supplier** | Подстрока в наименовании основного поставщика (ПОДОБНО). |
| **service** | Признак «Услуга» (true/false). Если не указан — возвращаются все. |

Все параметры опциональны. Если не указан ни один — возвращаются все элементы номенклатуры в виде дерева.
