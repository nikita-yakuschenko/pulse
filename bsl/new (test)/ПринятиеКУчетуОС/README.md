# Принятие к учёту ОС (GetFixedAssetAcceptance)

HTTP-сервис возвращает список документов **Принятие к учёту ОС** (Документ.ПринятиеКУчетуОС).

**Функция в конфигураторе:** `GetFixedAssetAcceptance`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| МОЛ | Материально-ответственное лицо |
| СуммаДокумента | Сумма документа |
| Проведен | Признак проведения |
| ОсновныеСредства | При `full=true` — массив строк ТЧ ОС |

### Строки ТЧ ОсновныеСредства (full=true)

| Поле | Описание |
|------|----------|
| ОсновноеСредство | Основное средство |
| НачальнаяСтоимость | Начальная стоимость |
| СрокПолезногоИспользования | Срок полезного использования (мес.) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего года |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ ОсновныеСредства |
