# Начисление амортизации (GetDepreciation)

HTTP-сервис возвращает список документов **Начисление амортизации** (Документ.НачислениеАмортизации).

**Функция в конфигураторе:** `GetDepreciation`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| СуммаДокумента | Итоговая сумма амортизации |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего года |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
