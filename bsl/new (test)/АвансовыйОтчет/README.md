# Авансовый отчет (GetAdvanceReports)

HTTP-сервис возвращает список документов **Авансовый отчет** (Документ.АвансовыйОтчет).

**Функция в конфигураторе:** `GetAdvanceReports`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| ФизическоеЛицо | Подотчётное физическое лицо |
| Подотчетное | Счёт подотчёта |
| СуммаДокумента | Выдано авансов |
| СуммаРасхода | Израсходовано |
| Ответственный | Ответственный |
| Проведен | Признак проведения |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **employee** | Подстрока в ФИО физического лица (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
