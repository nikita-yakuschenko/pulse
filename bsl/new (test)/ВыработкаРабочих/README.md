# Выработка рабочих (GetWorkerOutput)

HTTP-сервис возвращает обороты регистра **Выработка рабочих** (РегистрНакопления.ВыработкаРабочих.Обороты).

**Функция в конфигураторе:** `GetWorkerOutput`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Подразделение | Производственное подразделение |
| Сотрудник | Сотрудник |
| ТехОперация | Технологическая операция |
| КоличествоОборот | Количество за период |
| СуммаОборот | Сумма сдельной оплаты за период |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
