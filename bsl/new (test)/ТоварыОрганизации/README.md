# Товары организации (GetOrgGoods)

HTTP-сервис возвращает остатки по регистру накопления **Товары организаций** (РегистрНакопления.ТоварыОрганизаций.Остатки).

**Функция в конфигураторе:** `GetOrgGoods`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Номенклатура | Номенклатура |
| Характеристика | Характеристика номенклатуры |
| Количество | Количество (остаток) |
| Стоимость | Стоимость (остаток) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **item** | Подстрока в наименовании номенклатуры (пост-фильтр) |
