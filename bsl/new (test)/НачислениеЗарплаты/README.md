# Начисление зарплаты (GetPayrollAccrual)

HTTP-сервис возвращает список документов **Начисление зарплаты** (Документ.НачислениеЗарплаты).

**Функция в конфигураторе:** `GetPayrollAccrual`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Подразделение организации |
| СуммаДокумента | Итоговая сумма начислений |
| Проведен | Признак проведения |
| Начисления | При `full=true` — массив строк ТЧ Начисления |

### Строки ТЧ Начисления (full=true)

| Поле | Описание |
|------|----------|
| Сотрудник | Сотрудник |
| ВидРасчета | Вид начисления |
| Результат | Сумма начисления |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ Начисления |
