# План производства (GetProductionPlan)

HTTP-сервис возвращает список документов **План производства** (Документ.ПланПроизводства).

**Функция в конфигураторе:** `GetProductionPlan`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Производственное подразделение |
| НачалоПериода | Начало планового периода |
| КонецПериода | Конец планового периода |
| Проведен | Признак проведения |
| Продукция | При `full=true` — массив строк ТЧ Продукция |

### Строки ТЧ Продукция (full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Плановая продукция |
| Характеристика | Характеристика номенклатуры |
| КоличествоПлан | Плановое количество |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — конец текущего месяца |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ Продукция |
