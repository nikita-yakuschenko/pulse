# Остатки товаров в пути (GetGoodsInTransit)

HTTP-сервис возвращает остатки регистра **Товары организаций в пути** (РегистрНакопления.ТоварыОрганизацийВПути.Остатки).

**Функция в конфигураторе:** `GetGoodsInTransit`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Контрагент | Поставщик/перевозчик |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| КоличествоОстаток | Количество в пути |
| СуммаОстаток | Сумма товаров в пути |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
