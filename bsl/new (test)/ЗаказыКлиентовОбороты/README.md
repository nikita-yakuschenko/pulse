# Заказы клиентов — обороты (GetCustomerOrderTurnovers)

HTTP-сервис возвращает обороты регистра **Заказы покупателей** (РегистрНакопления.ЗаказыПокупателей.Обороты).

**Функция в конфигураторе:** `GetCustomerOrderTurnovers`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Контрагент | Покупатель |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| КоличествоПриход | Поступление заказов (количество) |
| КоличествоРасход | Исполнение заказов (количество) |
| СуммаПриход | Поступление заказов (сумма) |
| СуммаРасход | Исполнение заказов (сумма) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
