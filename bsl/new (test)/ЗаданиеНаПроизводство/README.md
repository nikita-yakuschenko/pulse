# Задание на производство (GetProductionTasks)

HTTP-сервис возвращает список документов **Задание на производство** (Документ.ЗаданиеНаПроизводство).

**Функция в конфигураторе:** `GetProductionTasks`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| Подразделение | Производственное подразделение |
| Ответственный | Ответственный |
| ДатаНачала | Плановая дата начала (dd.MM.yyyy) |
| ДатаОкончания | Плановая дата окончания (dd.MM.yyyy) |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| ВыпускТехПроцесс | `{Есть, Количество}` или массив строк ТЧ при `full=true` |

### Строка ТЧ ВыпускТехПроцесс (при full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Продукция |
| ХарактеристикаНоменклатуры | Характеристика |
| Количество | Плановое количество |
| ЕдиницаИзмерения | Единица измерения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **org** | Подстрока в организации (пост-фильтр) |
| **subdivision** | Подстрока в подразделении (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полная ТЧ ВыпускТехПроцесс |
