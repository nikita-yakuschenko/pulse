# Уценка товаров (GetMarkdown)

HTTP-сервис возвращает список документов **Уценка товаров** (Документ.УценкаТоваров).

**Функция в конфигураторе:** `GetMarkdown`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Склад | Склад |
| СуммаДокумента | Сумма уценки |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **warehouse** | Подстрока в наименовании склада (пост-фильтр) |
