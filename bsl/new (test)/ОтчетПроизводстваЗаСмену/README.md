# Отчёт производства за смену (GetShiftReport)

HTTP-сервис возвращает список документов **Отчёт производства за смену** (Документ.ОтчетПроизводстваЗаСмену).

**Функция в конфигураторе:** `GetShiftReport`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Производственное подразделение |
| Склад | Склад готовой продукции |
| СуммаДокумента | Плановая стоимость |
| Проведен | Признак проведения |
| Продукция | При `full=true` — массив строк ТЧ Продукция |

### Строки ТЧ Продукция (full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Выпущенная продукция |
| Количество | Количество |
| Стоимость | Стоимость |
| Спецификация | Спецификация |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ Продукция |
