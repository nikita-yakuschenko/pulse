# Должности (GetPositions)

HTTP-сервис возвращает список элементов справочника **Должности** (Справочник.Должности).

**Функция в конфигураторе:** `GetPositions`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование должности |
| Код | Код должности |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
