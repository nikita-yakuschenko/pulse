# Основные средства (GetFixedAssets)

HTTP-сервис возвращает остатки основных средств на указанную дату из регистра бухгалтерии **Хозрасчётный** по счёту 01 (ОС).

**Функция в конфигураторе:** `GetFixedAssets`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| ОсновноеСредство | Основное средство |
| Подразделение | Подразделение организации |
| Стоимость | Балансовая стоимость (остаток) |
| НакопленнаяАмортизация | Накопленная амортизация (остаток) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **name** | Подстрока в наименовании ОС (пост-фильтр) |
