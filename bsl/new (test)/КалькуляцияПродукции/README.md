# Калькуляция продукции (GetCosting)

HTTP-сервис возвращает список документов **Калькуляция продукции** (Документ.КалькуляцияПродукции).

**Функция в конфигураторе:** `GetCosting`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Номенклатура | Номенклатура (продукция) |
| Характеристика | Характеристика номенклатуры |
| Количество | Количество |
| СуммаДокумента | Сумма плановой себестоимости |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **item** | Подстрока в наименовании номенклатуры (пост-фильтр) |
