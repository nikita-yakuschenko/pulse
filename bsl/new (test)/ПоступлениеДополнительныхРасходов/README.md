# Поступление дополнительных расходов (GetAdditionalCosts)

HTTP-сервис возвращает список документов **Поступление дополнительных расходов** (Документ.ПоступлениеДополнительныхРасходов).

**Функция в конфигураторе:** `GetAdditionalCosts`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Контрагент | Контрагент-поставщик |
| Договор | Договор контрагента |
| СуммаДокумента | Сумма документа |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
