# Цены номенклатуры (GetPrices)

HTTP-сервис возвращает срез последних цен из регистра сведений **ЦеныНоменклатуры** на указанную дату.

**Функция в конфигураторе:** `GetPrices`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| ТипЦен | Тип цен (прайс-лист) |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| Цена | Цена |
| Валюта | Валюта |
| ЕдиницаИзмерения | Единица измерения |
| Период | Дата последнего изменения цены (dd.MM.yyyy) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (по умолчанию — текущая дата) |
| **pricetype** | Подстрока в типе цен (пост-фильтр) |
| **item** | Подстрока в номенклатуре (пост-фильтр) |
