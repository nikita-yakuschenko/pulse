# Взаиморасчеты с контрагентами (GetDebts)

HTTP-сервис возвращает остатки по регистру накопления **ВзаиморасчетыСКонтрагентами** на указанную дату.

**Функция в конфигураторе:** `GetDebts`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Контрагент | Контрагент |
| Организация | Организация |
| ДоговорКонтрагента | Договор |
| Сумма | Сумма взаиморасчетов (остаток) |

> Положительное значение — долг контрагента перед организацией; отрицательное — долг организации перед контрагентом.

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (по умолчанию — текущая дата) |
| **contractor** | Подстрока в контрагенте (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **contract** | Подстрока в договоре (пост-фильтр) |
