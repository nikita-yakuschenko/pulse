# Единицы измерения (GetUnits)

HTTP-сервис возвращает список записей справочника **Классификатор единиц измерения** (Справочник.КлассификаторЕдиницИзмерения).

**Функция в конфигураторе:** `GetUnits`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Краткое наименование (шт, кг, м, …) |
| НаименованиеПолное | Полное наименование |
| Код | Код ОКЕИ |
| МеждународноеСокращение | Международное сокращение |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
