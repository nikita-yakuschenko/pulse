# Статьи ДДС (GetCashFlowItems)

HTTP-сервис возвращает список записей справочника **Статьи движения денежных средств** (Справочник.СтатьиДвиженияДенежныхСредств).

**Функция в конфигураторе:** `GetCashFlowItems`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование статьи |
| Код | Код |
| ВидДвижения | Вид движения (Поступление / Выбытие / Прочее) |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
| **kind** | Подстрока в виде движения (пост-фильтр) |
