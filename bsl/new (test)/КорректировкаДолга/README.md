# Корректировка долга (GetDebtAdjustments)

HTTP-сервис возвращает список документов **Корректировка долга** (Документ.КорректировкаДолга).

**Функция в конфигураторе:** `GetDebtAdjustments`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| Контрагент | Контрагент-дебитор |
| ДоговорКонтрагента | Договор дебитора |
| КонтрагентКредитор | Контрагент-кредитор (зачёт долга) |
| ДоговорКредитора | Договор кредитора |
| СуммаДокумента | Сумма |
| ВидОперации | Вид операции |
| Проведен | Признак проведения |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **contractor** | Подстрока в контрагенте-дебиторе (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
