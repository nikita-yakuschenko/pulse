# Графики работы (GetWorkSchedules)

HTTP-сервис возвращает список элементов справочника **Графики работы** (Справочник.ГрафикиРаботы).

**Функция в конфигураторе:** `GetWorkSchedules`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование графика |
| Код | Код |
| КоличествоЧасов | Часов в периоде |
| КоличествоДней | Дней в периоде |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
