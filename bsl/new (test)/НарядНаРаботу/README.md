# Наряд на работу (GetWorkOrders)

HTTP-сервис возвращает список документов **Наряд на работу** (Документ.НарядНаРаботу).

**Функция в конфигураторе:** `GetWorkOrders`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Подразделение организации |
| СуммаДокумента | Сумма документа |
| Проведен | Признак проведения |
| Работы | При `full=true` — массив строк ТЧ Работы |

### Строки ТЧ Работы (full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Вид работы (номенклатура) |
| Количество | Количество |
| Цена | Цена |
| Сумма | Сумма |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ Работы |
