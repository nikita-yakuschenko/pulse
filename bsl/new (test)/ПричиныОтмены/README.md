# Причины отмены (GetCancellationReasons)

HTTP-сервис возвращает список элементов справочника **Причины отмены** (Справочник.ПричиныОтмены).

**Функция в конфигураторе:** `GetCancellationReasons`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование причины |
| Код | Код |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
