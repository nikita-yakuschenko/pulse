# Корректировка стоимости товаров (GetCostAdjustment)

HTTP-сервис возвращает список документов **Корректировка стоимости товаров** (Документ.КорректировкаСтоимостиТоваров).

**Функция в конфигураторе:** `GetCostAdjustment`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| СуммаДокумента | Сумма корректировки |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
