# Акт об установленном расхождении (GetShortageAct)

HTTP-сервис возвращает список документов **Акт об установленном расхождении** (Документ.АктОбУстановленномРасхождении).

**Функция в конфигураторе:** `GetShortageAct`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Контрагент | Поставщик/перевозчик |
| СуммаДокумента | Сумма расхождения |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
