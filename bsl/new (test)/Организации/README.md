# Организации (GetOrganizations)

HTTP-сервис возвращает список записей справочника **Организации** (Справочник.Организации).

**Функция в конфигураторе:** `GetOrganizations`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Краткое наименование |
| НаименованиеПолное | Полное наименование |
| ИНН | ИНН |
| КПП | КПП |
| Префикс | Префикс нумерации документов |
| ЮрФизЛицо | Юридическое или физическое лицо |
| БанковскийСчет | Основной банковский счёт |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
| **inn** | Подстрока в ИНН (пост-фильтр) |
