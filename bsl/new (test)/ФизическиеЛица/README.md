# Физические лица (GetPersons)

HTTP-сервис возвращает список записей справочника **Физические лица** (Справочник.ФизическиеЛица).

**Функция в конфигураторе:** `GetPersons`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| ФИО | Полное имя |
| Код | Код |
| ДатаРождения | Дата рождения (dd.MM.yyyy) |
| Пол | Пол |
| ИНН | ИНН |
| СНИЛС | СНИЛС |
| Гражданство | Гражданство |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в ФИО (пост-фильтр) |
