# Выпуск продукции (GetProduction)

HTTP-сервис возвращает список документов **Выпуск продукции** (Документ.ВыпускПродукции).

**Функция в конфигураторе:** `GetProduction`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| Склад | Склад готовой продукции |
| Подразделение | Производственное подразделение |
| Ответственный | Ответственный |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| Продукция | `{Есть, Количество}` или массив строк ТЧ при `full=true` |

### Строка ТЧ Продукция (при full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Наименование продукции |
| ХарактеристикаНоменклатуры | Характеристика |
| Количество | Количество |
| ЕдиницаИзмерения | Единица измерения |
| Спецификация | Спецификация |
| Заказ | Заказ (покупателя или на производство) |
| НоменклатурнаяГруппа | Номенклатурная группа |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **org** | Подстрока в организации (пост-фильтр) |
| **warehouse** | Подстрока в складе (пост-фильтр) |
| **subdivision** | Подстрока в подразделении (пост-фильтр) |
| **responsible** | Подстрока в ответственном (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полная ТЧ Продукция |
