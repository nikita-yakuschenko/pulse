# Технологические операции (GetTechOperations)

HTTP-сервис возвращает список элементов справочника **Технологические операции** (Справочник.ТехнологическиеОперации).

**Функция в конфигураторе:** `GetTechOperations`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование операции |
| Код | Код |
| ЕдиницаНормирования | Единица нормирования трудозатрат |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
