# Платежное поручение входящее (GetIncomingPayments)

HTTP-сервис возвращает список документов **Платежное поручение входящее** (Документ.ПлатежноеПоручениеВходящее).

**Функция в конфигураторе:** `GetIncomingPayments`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация-получатель |
| Контрагент | Плательщик |
| СуммаДокумента | Сумма |
| НазначениеПлатежа | Назначение платежа |
| ДоговорКонтрагента | Договор |
| СтатьяДДС | Статья движения денежных средств |
| ВидОперации | Вид операции |
| Ответственный | Ответственный |
| Проведен | Признак проведения |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **contractor** | Подстрока в плательщике (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
