# Валюты (GetCurrencies)

HTTP-сервис возвращает список элементов справочника **Валюты** (Справочник.Валюты).

**Функция в конфигураторе:** `GetCurrencies`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Краткое наименование (RUB, USD и т.д.) |
| Код | Числовой код валюты |
| НаименованиеПолное | Полное наименование |
| СимвольныйКод | Символьный код (ISO 4217) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
