# Передача в производство (GetTransferToProduction)

HTTP-сервис возвращает список документов **Передача в производство** (Документ.ПередачаВПроизводство).

**Функция в конфигураторе:** `GetTransferToProduction`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Подразделение-получатель |
| Склад | Склад-отправитель |
| Проведен | Признак проведения |
| Материалы | При `full=true` — массив строк ТЧ Материалы |

### Строки ТЧ Материалы (full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Передаваемый материал |
| Количество | Количество |
| Стоимость | Стоимость |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
| **full** | `true` / `1` / `да` — вернуть ТЧ Материалы |
