# Номенклатурные группы (GetNomGroups)

HTTP-сервис возвращает список записей справочника **Номенклатурные группы** (Справочник.НоменклатурныеГруппы).

**Функция в конфигураторе:** `GetNomGroups`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование группы |
| Код | Код |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
