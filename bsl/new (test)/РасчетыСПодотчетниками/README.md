# Расчёты с подотчётниками (GetAccountablePersons)

HTTP-сервис возвращает остатки регистра **Расчёты с подотчётными лицами** (РегистрНакопления.РасчетыСПодотчетниками.Остатки).

**Функция в конфигураторе:** `GetAccountablePersons`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| ФизическоеЛицо | Подотчётное лицо |
| СуммаОстаток | Остаток задолженности |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
