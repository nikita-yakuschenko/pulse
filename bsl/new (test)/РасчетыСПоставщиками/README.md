# Расчеты с поставщиками (GetSupplierDebts)

HTTP-сервис возвращает остатки регистра **Расчеты с поставщиками** (РегистрНакопления.РасчетыСПоставщиками.Остатки).

**Функция в конфигураторе:** `GetSupplierDebts`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Контрагент | Поставщик |
| Договор | Договор |
| СуммаОстаток | Остаток задолженности (в валюте договора) |
| СуммаРеглОстаток | Остаток задолженности (в регл. валюте) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
