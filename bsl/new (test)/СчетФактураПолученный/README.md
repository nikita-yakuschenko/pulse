# Счёт-фактура полученный (GetVATInvoicesIn)

HTTP-сервис возвращает список документов **Счёт-фактура полученный** (Документ.СчетФактураПолученный).

**Функция в конфигураторе:** `GetVATInvoicesIn`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Контрагент | Поставщик |
| ДокументОснование | Документ-основание (поступление) |
| СуммаДокумента | Сумма |
| СуммаНДС | Сумма НДС |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
