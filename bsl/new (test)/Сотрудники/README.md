# Сотрудники (GetEmployees)

HTTP-сервис возвращает список элементов справочника **Сотрудники** (Справочник.Сотрудники).

**Функция в конфигураторе:** `GetEmployees`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | ФИО сотрудника |
| ФизическоеЛицо | Ссылка на физическое лицо |
| Организация | Организация |
| Подразделение | Подразделение |
| Должность | Должность |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
