# Отражение зарплаты в регл. учёте (GetPayrollAccounting)

HTTP-сервис возвращает список документов **Отражение зарплаты в регл. учёте** (Документ.ОтражениеЗарплатыВРеглУчете).

**Функция в конфигураторе:** `GetPayrollAccounting`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| СуммаДокумента | Сумма проводок |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
