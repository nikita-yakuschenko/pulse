# Банковские счета организаций (GetOrgBankAccounts)

HTTP-сервис возвращает список записей справочника **Банковские счета** (Справочник.БанковскиеСчета) для организаций.

**Функция в конфигураторе:** `GetOrgBankAccounts`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование счёта |
| Код | Код |
| Организация | Организация-владелец |
| Банк | Банк |
| НомерСчета | Номер расчётного счёта |
| Валюта | Валюта счёта |
| НазначениеСчета | Назначение счёта |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **bank** | Подстрока в наименовании банка (пост-фильтр) |
| **account** | Подстрока в номере счёта (пост-фильтр) |
| **currency** | Подстрока в наименовании валюты (пост-фильтр) |
