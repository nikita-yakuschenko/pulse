# Закупки (GetPurchasesSummary)

HTTP-сервис возвращает обороты по регистру накопления **Закупки** (РегистрНакопления.Закупки.Обороты).

**Функция в конфигураторе:** `GetPurchasesSummary`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Контрагент | Поставщик |
| Организация | Организация |
| Номенклатура | Номенклатура |
| Количество | Количество (оборот) |
| Стоимость | Стоимость (оборот) |
| СтоимостьБезНДС | Стоимость без НДС (оборот) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего года |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
