# Типы цен номенклатуры (GetPriceTypes)

HTTP-сервис возвращает список записей справочника **Типы цен номенклатуры** (Справочник.ТипыЦенНоменклатуры).

**Функция в конфигураторе:** `GetPriceTypes`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование типа цен |
| Код | Код |
| Валюта | Валюта |
| ФормулаЦены | Формула расчёта цены |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
