# Реализация товаров и услуг (GetSales)

HTTP-сервис возвращает список документов **Реализация товаров и услуг** (Документ.РеализацияТоваровУслуг).

**Функция в конфигураторе:** `GetSales`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Контрагент | Покупатель |
| Организация | Организация |
| Склад | Склад отгрузки |
| Ответственный | Ответственный |
| СуммаДокумента | Сумма |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| ДоговорКонтрагента | Договор |
| Сделка | Заказ покупателя / счёт-основание |
| ВидОперации | Вид операции |
| Товары | `{Есть, Количество}` или массив строк при `full=true` |
| Услуги | `{Есть, Количество}` или массив строк при `full=true` |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **contractor** | Подстрока в покупателе (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **warehouse** | Подстрока в складе (пост-фильтр) |
| **responsible** | Подстрока в ответственном (пост-фильтр) |
| **comment** | Подстрока в комментарии (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полные ТЧ Товары и Услуги |
