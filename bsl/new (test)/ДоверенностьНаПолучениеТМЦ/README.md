# Доверенность на получение ТМЦ (GetProxy)

HTTP-сервис возвращает список документов **Доверенность на получение ТМЦ** (Документ.ДоверенностьНаПолучениеТМЦ).

**Функция в конфигураторе:** `GetProxy`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Контрагент | Контрагент-получатель |
| ФизическоеЛицо | Доверенное лицо |
| ДатаОкончания | Срок действия доверенности |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
