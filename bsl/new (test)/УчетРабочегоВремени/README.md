# Учёт рабочего времени (GetTimesheet)

HTTP-сервис возвращает обороты регистра **Учёт рабочего времени** (РегистрНакопления.УчетРабочегоВремени.Обороты).

**Функция в конфигураторе:** `GetTimesheet`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Сотрудник | Сотрудник |
| ВидВремени | Вид использования рабочего времени |
| ЧасыОборот | Часы за период |
| ДниОборот | Дни за период |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
