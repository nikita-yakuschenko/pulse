# Счёт-фактура выданная (GetVATInvoicesOut)

HTTP-сервис возвращает список документов **Счёт-фактура выданный** (Документ.СчетФактураВыданный).

**Функция в конфигураторе:** `GetVATInvoicesOut`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Контрагент | Покупатель |
| ДокументОснование | Документ-основание (реализация / акт) |
| СуммаДокумента | Сумма документа |
| СуммаНДС | Сумма НДС |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **contractor** | Подстрока в наименовании контрагента (пост-фильтр) |
