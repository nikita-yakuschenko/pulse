# Банки (GetBanks)

HTTP-сервис возвращает список записей справочника **Банки** (Справочник.Банки).

**Функция в конфигураторе:** `GetBanks`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование банка |
| Код | Код |
| НомерКорСчета | Номер корреспондентского счёта |
| КодБИК | БИК |
| Город | Город |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
| **bic** | Подстрока в БИК (пост-фильтр) |
