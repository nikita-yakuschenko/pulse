# Расчёты с работниками (GetEmployeeSettlements)

HTTP-сервис возвращает остатки регистра **Расчёты с работниками** (РегистрНакопления.РасчетыСРаботниками.Остатки).

**Функция в конфигураторе:** `GetEmployeeSettlements`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Сотрудник | Сотрудник |
| СуммаОстаток | Остаток задолженности по зарплате |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
