# Подразделения организаций (GetDivisions)

HTTP-сервис возвращает список записей справочника **Подразделения организаций** (Справочник.ПодразделенияОрганизаций).

**Функция в конфигураторе:** `GetDivisions`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование подразделения |
| Организация | Организация-владелец |
| Код | Код |
| Родитель | Родительское подразделение |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
