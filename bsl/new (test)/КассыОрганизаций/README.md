# Кассы организаций (GetCashRegisters)

HTTP-сервис возвращает список элементов справочника **Кассы организаций** (Справочник.КассыОрганизаций).

**Функция в конфигураторе:** `GetCashRegisters`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование кассы |
| Код | Код |
| Организация | Организация |
| Валюта | Валюта кассы |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
