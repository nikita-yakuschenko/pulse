# Себестоимость товаров (GetGoodsCost)

HTTP-сервис возвращает остатки регистра **Себестоимость товаров** (РегистрНакопления.СебестоимостьТоваров.Остатки).

**Функция в конфигураторе:** `GetGoodsCost`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| КоличествоОстаток | Остаток по количеству |
| СтоимостьОстаток | Остаток по себестоимости |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
