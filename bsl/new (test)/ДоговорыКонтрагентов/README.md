# Договоры контрагентов (GetContracts)

HTTP-сервис возвращает список записей справочника **Договоры контрагентов** (Справочник.ДоговорыКонтрагентов).

**Функция в конфигураторе:** `GetContracts`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование договора |
| Контрагент | Контрагент-владелец договора |
| Организация | Организация |
| ВидДоговора | Вид договора (С покупателем, С поставщиком, …) |
| Валюта | Валюта договора |
| СуммаДоговора | Сумма договора |
| ДатаНачала | Дата начала (dd.MM.yyyy) |
| ДатаОкончания | Дата окончания (dd.MM.yyyy) |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании договора (пост-фильтр) |
| **contractor** | Подстрока в контрагенте (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **kind** | Подстрока в виде договора (пост-фильтр) |
