# Незавершенное производство (GetWIP)

HTTP-сервис возвращает остатки по регистру накопления **НезавершенноеПроизводство** на указанную дату.

**Функция в конфигураторе:** `GetWIP`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номенклатура | Номенклатура (продукция в производстве) |
| ХарактеристикаНоменклатуры | Характеристика |
| Организация | Организация |
| Подразделение | Производственное подразделение |
| Количество | Остаток количества |
| Стоимость | Остаток стоимости |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (по умолчанию — текущая дата) |
| **org** | Подстрока в организации (пост-фильтр) |
| **subdivision** | Подстрока в подразделении (пост-фильтр) |
| **item** | Подстрока в номенклатуре (пост-фильтр) |
