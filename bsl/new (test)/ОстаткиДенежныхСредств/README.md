# Остатки денежных средств (GetCashBalances)

HTTP-сервис возвращает остатки по регистру накопления **Денежные средства** (РегистрНакопления.ДенежныеСредства.Остатки).

**Функция в конфигураторе:** `GetCashBalances`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| БанковскийСчет | Банковский счёт / касса |
| Валюта | Валюта |
| Сумма | Остаток в рублях |
| СуммаВалютная | Остаток в валюте |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
