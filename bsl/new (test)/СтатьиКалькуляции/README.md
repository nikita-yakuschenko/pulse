# Статьи калькуляции (GetCostingItems)

HTTP-сервис возвращает список элементов справочника **Статьи калькуляции** (Справочник.СтатьиКалькуляции).

**Функция в конфигураторе:** `GetCostingItems`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование статьи |
| Код | Код |
| ВидЗатрат | Вид затрат (перечисление) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
