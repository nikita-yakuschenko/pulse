# Инвентаризация товаров на складе (GetInventory)

HTTP-сервис возвращает список документов **Инвентаризация товаров на складе** (Документ.ИнвентаризацияТоваровНаСкладе).

**Функция в конфигураторе:** `GetInventory`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| Склад | Склад |
| Ответственный | Ответственный |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| Товары | `{Есть, Количество}` или массив строк ТЧ при `full=true` |

### Строка ТЧ Товары (при full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| КоличествоУчет | Количество по учёту |
| КоличествоФакт | Фактическое количество |
| Отклонение | КоличествоФакт − КоличествоУчет |
| ЕдиницаИзмерения | Единица измерения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **org** | Подстрока в организации (пост-фильтр) |
| **warehouse** | Подстрока в складе (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полная ТЧ Товары с отклонениями |
