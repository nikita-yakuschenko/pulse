# Возврат товаров от покупателя (GetCustomerReturns)

HTTP-сервис возвращает список документов **Возврат товаров от покупателя** (Документ.ВозвратТоваровОтПокупателя).

**Функция в конфигураторе:** `GetCustomerReturns`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Контрагент | Покупатель |
| Организация | Организация |
| Склад | Склад |
| Ответственный | Ответственный |
| СуммаДокумента | Сумма |
| ДоговорКонтрагента | Договор |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| Товары | `{Есть, Количество}` или массив строк ТЧ при `full=true` |

### Строка ТЧ Товары (при full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Наименование |
| ХарактеристикаНоменклатуры | Характеристика |
| Количество | Количество |
| Цена | Цена |
| Сумма | Сумма строки |
| ЕдиницаИзмерения | Единица измерения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **contractor** | Подстрока в покупателе (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **warehouse** | Подстрока в складе (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полная ТЧ Товары |
