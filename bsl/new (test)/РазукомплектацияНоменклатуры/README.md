# Разукомплектация номенклатуры (GetDisassembly)

HTTP-сервис возвращает список документов **Разукомплектация номенклатуры** (Документ.РазукомплектацияНоменклатуры).

**Функция в конфигураторе:** `GetDisassembly`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация |
| Склад | Склад |
| Номенклатура | Разукомплектуемая номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| Количество | Количество |
| ЕдиницаИзмерения | Единица измерения |
| Ответственный | Ответственный |
| Проведен | Признак проведения |
| Комментарий | Комментарий |
| Компоненты | `{Есть, Количество}` или массив строк ТЧ при `full=true` |

### Строка ТЧ Компоненты (при full=true)

| Поле | Описание |
|------|----------|
| Номенклатура | Компонент |
| ХарактеристикаНоменклатуры | Характеристика |
| Количество | Количество |
| ЕдиницаИзмерения | Единица измерения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **org** | Подстрока в организации (пост-фильтр) |
| **warehouse** | Подстрока в складе (пост-фильтр) |
| **approved** | `true/1/ДА` — только проведённые |
| **full** | `true/1/ДА` — полная ТЧ Компоненты |
