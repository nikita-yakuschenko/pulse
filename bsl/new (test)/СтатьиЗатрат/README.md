# Статьи затрат (GetCostItems)

HTTP-сервис возвращает список записей справочника **Статьи затрат** (Справочник.СтатьиЗатрат).

**Функция в конфигураторе:** `GetCostItems`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Наименование статьи |
| Код | Код |
| ВидРасходов | Вид расходов (НУ) |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
