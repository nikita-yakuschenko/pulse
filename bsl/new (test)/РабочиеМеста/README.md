# Рабочие места (GetWorkplaces)

HTTP-сервис возвращает список элементов справочника **Рабочие места** (Справочник.РабочиеМеста).

**Функция в конфигураторе:** `GetWorkplaces`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование рабочего места |
| Код | Код |
| Подразделение | Производственное подразделение |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
| **division** | Подстрока в наименовании подразделения (пост-фильтр) |
