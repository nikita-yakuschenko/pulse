# Платежное поручение исходящее (GetPaymentOrders)

HTTP-сервис возвращает список документов **Платежное поручение исходящее** (Документ.ПлатежноеПоручениеИсходящее).

**Функция в конфигураторе:** `GetPaymentOrders`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Номер | Номер документа |
| Дата | Дата (dd.MM.yyyy) |
| Организация | Организация-плательщик |
| Контрагент | Получатель платежа |
| СуммаДокумента | Сумма платежа |
| НазначениеПлатежа | Назначение платежа |
| ДокументОснование | Документ-основание |
| ДоговорКонтрагента | Договор |
| СтатьяДДС | Статья движения денежных средств |
| Оплачено | Признак оплаты |
| Проведен | Признак проведения |
| Ответственный | Ответственный |
| Комментарий | Комментарий |
| ВидОперации | Вид операции |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from**, **to** | Период по дате |
| **code** | Подстрока в номере (ПОДОБНО) |
| **contractor** | Подстрока в контрагенте (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **paid** | `true/1/ДА` — только оплаченные |
| **approved** | `true/1/ДА` — только проведённые |
