# Продажи (GetSalesSummary)

HTTP-сервис возвращает обороты из регистра накопления **Продажи** за указанный период.

**Функция в конфигураторе:** `GetSalesSummary`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Контрагент | Покупатель |
| Организация | Организация |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| НоменклатурнаяГруппа | Номенклатурная группа |
| Количество | Оборот количества |
| СуммаВыручки | Выручка с НДС |
| СуммаВыручкиБезНДС | Выручка без НДС |
| Себестоимость | Себестоимость |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **from** | Начало периода (по умолчанию — начало текущего года) |
| **to** | Конец периода (по умолчанию — текущая дата) |
| **contractor** | Подстрока в контрагенте (пост-фильтр) |
| **org** | Подстрока в организации (пост-фильтр) |
| **item** | Подстрока в номенклатуре (пост-фильтр) |
| **nomgroup** | Подстрока в номенклатурной группе (пост-фильтр) |
