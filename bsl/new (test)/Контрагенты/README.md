# Контрагенты (GetContractors)

HTTP-сервис возвращает список записей справочника **Контрагенты** (Справочник.Контрагенты).

**Функция в конфигураторе:** `GetContractors`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Наименование | Краткое наименование |
| НаименованиеПолное | Полное наименование |
| ИНН | ИНН |
| КПП | КПП |
| ЮрФизЛицо | Юридическое или физическое лицо |
| ЭтоПоставщик | Признак поставщика |
| ЭтоПокупатель | Признак покупателя |
| Телефоны | Телефоны |
| Email | Адрес электронной почты |
| Комментарий | Комментарий |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **name** | Подстрока в наименовании (пост-фильтр) |
| **inn** | Подстрока в ИНН (пост-фильтр) |
| **supplier** | `true/1/ДА` — только поставщики (SQL-фильтр) |
| **buyer** | `true/1/ДА` — только покупатели (SQL-фильтр) |
