# Партии товаров на складах (GetGoodsBatches)

HTTP-сервис возвращает остатки регистра **Партии товаров на складах** (РегистрНакопления.ПартииТоваровНаСкладах.Остатки).

**Функция в конфигураторе:** `GetGoodsBatches`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Склад | Склад |
| Номенклатура | Номенклатура |
| ХарактеристикаНоменклатуры | Характеристика |
| КоличествоОстаток | Остаток по количеству |
| СтоимостьОстаток | Остаток по стоимости |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **date** | Дата среза (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
| **warehouse** | Подстрока в наименовании склада (пост-фильтр) |
