# Перечисление зарплаты в банк (GetPayrollBankTransfer)

HTTP-сервис возвращает список документов **Перечисление зарплаты в банк** (Документ.ПеречислениеЗарплатыВБанк).

**Функция в конфигураторе:** `GetPayrollBankTransfer`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление документа |
| Дата | Дата документа |
| Номер | Номер документа |
| Организация | Организация |
| Подразделение | Подразделение |
| БанковскийСчет | Банковский счёт организации |
| СуммаДокумента | Сумма перечисления |
| Проведен | Признак проведения |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
