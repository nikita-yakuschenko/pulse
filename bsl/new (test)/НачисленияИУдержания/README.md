# Начисления и удержания (GetPayrollAccruals)

HTTP-сервис возвращает обороты регистра расчёта **Начисления и удержания** (РегистрРасчета.НачисленияИУдержания.Обороты).

**Функция в конфигураторе:** `GetPayrollAccruals`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Организация | Организация |
| Сотрудник | Сотрудник |
| ВидРасчета | Вид расчёта (начисление/удержание) |
| РезультатОборот | Сумма за период |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **datefrom** | Начало периода (дд.ММ.гггг или гггг-ММ-дд). По умолчанию — начало текущего месяца |
| **dateto** | Конец периода. По умолчанию — текущая дата |
| **org** | Подстрока в наименовании организации (пост-фильтр) |
