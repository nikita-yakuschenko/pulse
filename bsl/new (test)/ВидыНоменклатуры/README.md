# Виды номенклатуры (GetItemTypes)

HTTP-сервис возвращает список элементов справочника **Виды номенклатуры** (Справочник.ВидыНоменклатуры).

**Функция в конфигураторе:** `GetItemTypes`.

---

## Поля ответа

| Поле | Описание |
|------|----------|
| Ссылка | Строковое представление элемента |
| Наименование | Наименование вида номенклатуры |
| Код | Код |
| ВидНоменклатуры | Тип/вид (перечисление) |

---

## Параметры запроса

| Параметр | Назначение |
|----------|------------|
| **search** | Подстрока в наименовании (пост-фильтр) |
